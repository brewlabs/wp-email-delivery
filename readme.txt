== Installation ==

Installing "WP Email Delivery" can be done either by searching for "WP Email Delivery" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
1. 'WP Email Delivery' will now show under 'Settings'
1. Enter your license key to start sending
